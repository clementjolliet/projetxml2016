Une fois le serveur lancé avec node server.js et la base de données Exist démarré, on peut lancer une recherche dans la base XML via un appel HTTP en GET sur le serveur (http://localhost:1337/). Les parametres sont :
- search : la chaine de caractères de recherche
- start : l'item dans le fichier xml à partir duquel on commence à chercher (par défaut 0)
- length : le nombre d'item sur lesquels on va chercher dans le fichier xml
L'appel HTTP retourne toutes les données sur les monuments trouvés en JSON
